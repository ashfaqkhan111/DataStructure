3.4.3	Question


Answer : 
        
        Yes, a class can have more than one constructor. This is called constructor overloading, where multiple constructors 
        have the same name but different parameter lists to initialize objects in different ways.

        e.g 
            class Student {

    String name;
    String nim;
    float gpa;

    // Constructor 1 (no parameter)
    Student() {
        name = "Unknown";
        nim = "000";
        gpa = 0;
    }

    // Constructor 2 (with parameters)
    Student(String name, String nim, float gpa) {
        this.name = name;
        this.nim = nim;
        this.gpa = gpa;
    }
}
