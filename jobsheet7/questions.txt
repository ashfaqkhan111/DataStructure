Q1 : Show the program code in which runs the divide process
Answer : the divide process happens in the sort method where the array is split into smaller parts

```java
private void sort(int listStd[], int left, int right){
    if(left < right){
        int mid = (left + right)/2;
        sort(listStd, left, mid);
        sort(listStd, mid+1, right);
    }
}
```

Q2 : Show the program code in which runs the conquer process
Answer : the conquer process happens in the merge method where the divided parts are combined back in sorted order

```java
public void merge(int[] listStd, int left, int mid, int right){
    int temp[] = new int[listStd.length];

    for (int i = left; i <= right; i++){
        temp[i] = listStd[i];
    }

    int a = left;
    int b = mid + 1;
    int c = left;

    while (a <= mid && b <= right){
        if (temp[a] <= temp[b]){
            listStd[c] = temp[a];
            a++;
        } else {
            listStd[c] = temp[b];
            b++;
        }
        c++;
    }

    while (a <= mid){
        listStd[c] = temp[a];
        a++;
        c++;
    }

    while (b <= right){
        listStd[c] = temp[b];
        b++;
        c++;
    }
}
```

Q3 : If inserted NIM data is not sorted will the program give the correct result why
Answer : sequential search will still give correct results because it checks all elements one by one. binary search will not give correct results because it requires sorted data

If inserted data is sorted from largest to smallest and searching 20210 the binary search will not return the correct result because it assumes ascending order

to fix this for descending order the comparison must be reversed

```java
public int findBinarySearch(int search, int left, int right){
    if (left <= right){
        int mid = (left + right)/2;

        if (search == listStd[mid].nim){
            return mid;
        } else if (listStd[mid].nim < search){
            return findBinarySearch(search, left, mid - 1);
        } else {
            return findBinarySearch(search, mid + 1, right);
        }
    }
    return -1;
}
```
