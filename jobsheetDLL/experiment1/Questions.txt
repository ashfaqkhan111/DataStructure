1. Explain the difference between a singly linked list and a doubly linked list.

A singly linked list stores only one link called next, which points to the next node. A doubly linked list stores two links, next and prev, so nodes can move forward and backward.

2. Observe the Node class, which contains the attributes next and prev. What are the purposes of these attributes?

The next attribute stores the address of the next node, while the prev attribute stores the address of the previous node. These attributes connect nodes together in a doubly linked list.
